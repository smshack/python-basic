# 불과 비교, 논리 연산자 알아보기
```
참(True), 거짓(False)을 나타내는 불(boolean)
두 값의 관계를 판단하는 비교 연산자와 두 값의 논릿값을 판단하는 논리 연산자
```

## 비교 연산자 - 부등호
- 숫자, 문자열 비교

## 객체가 같은지 다른지 비교
```
is 와 is not
== , != 는 값 자체를 비교하고
is, is not 은 객체를 비교함
```

## 논리 연산자
```
a and b
a or b
not x
```

## 단락 평가
- 첫번째 값만으로 결과가 확실할 때 두 번째 값은 확인하지 않는 방법
- 비교할 때 앞에 로직이 간단한것 뒤에 로직이 큰걸 놓는 것이 좋음
